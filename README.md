## 📖 About the Project

**FlappyMC** is a custom 2D arcade game that blends the classic *Flappy Bird* gameplay with a *Minecraft-inspired* aesthetic.  

The project is built entirely in Python and showcases:
- 🎮 Smooth arcade-style mechanics  
- 🌍 Procedural obstacle generation  
- 🧑‍🎨 Live Minecraft skin integration via API  
- ⚡ Lightweight and easy-to-run game loop  

---

## 🚀 Features

- Randomly generated pipes/obstacles  
- Player skins pulled dynamically from online APIs  
- Score tracking system  
- Simple and responsive controls  
- Clean modular Python structure  

---

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/YourUsername/FlappyMC.git
cd FlappyMC
```

### 2. Install dependencies
Make sure you have Python 3 installed, then run:
```bash
pip install pygame requests
```

### 3. Run the game
```bash
python flappymc.py
```

## 🎮 Controls

- **Spacebar / Click** → Jump
